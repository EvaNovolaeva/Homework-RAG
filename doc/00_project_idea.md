# Идея проекта

## Проблема

Подростки активно используют социальные сети, и это может влиять на их психологическое состояние, уровень тревожности, стресс, качество сна и общее самочувствие. При этом данные по этой теме часто представлены в виде таблиц и отдельных исследований, поэтому пользователю сложно быстро находить ответы на конкретные вопросы и интерпретировать связи между признаками.

## Цель проекта

Цель проекта - создать RAG-ассистента, который помогает отвечать на вопросы о влиянии социальных сетей на ментальное здоровье подростков на основе выбранного датасета. Пользователь сможет задавать вопросы на естественном языке, а система будет находить релевантные записи и признаки в данных, использовать их как контекст и формировать понятный ответ.

## Для кого проект

Проект предназначен для студентов, начинающих аналитиков, исследователей и всех, кто хочет изучать влияние социальных сетей на подростковое ментальное здоровье. Также проект может быть полезен преподавателям и пользователям, которым нужен удобный интерфейс для быстрого анализа темы без ручного просмотра таблиц и описаний датасета.

## Какие данные используете

В проекте планируется использовать публичный датасет с Kaggle Social Media Impact on Teen Mental Health: https://www.kaggle.com/datasets/algozee/teenager-menthal-healy. По описанию датасет посвящён изучению того, как использование социальных сетей связано с ментальным здоровьем подростков, и включает такие показатели, как время в социальных сетях, сон, стресс, тревожность и другие поведенческие и психологические характеристики.

В качестве источника данных используется Kaggle. Масштаб данных — более 1000 записей, что соответствует требованию по объёму для учебного проекта. На следующих этапах данные будут подготовлены для использования в RAG-пайплайне: строки датасета могут быть преобразованы в текстовые описания, после чего будут разбиты на фрагменты, проиндексированы и использованы как база знаний для ответов на вопросы.

## Критерий успеха

Проект можно считать успешным, если система способна отвечать на типовые вопросы пользователя по датасету в понятной и логичной форме, опираясь на извлечённые из данных релевантные фрагменты. Также важным критерием успеха является то, что пользователь может получить ответ быстрее, чем при самостоятельном поиске по таблице или ручном анализе датасета.

Дополнительно успехом считается наличие работающего прототипа, который:
- загружает и обрабатывает данные;
- преобразует данные в формат, пригодный для поиска;
- находит релевантную информацию по запросу;
- использует найденный контекст при генерации ответа.

## Как проверить

Проверка проекта будет выполняться с помощью набора тестовых вопросов по датасету. Например: какие факторы сильнее всего связаны со стрессом, как соотносятся время в социальных сетях и качество сна, какие признаки чаще встречаются у подростков с более высоким уровнем тревожности.

Если система на такие вопросы выдаёт осмысленные ответы, использует релевантные данные и может объяснить вывод на основе найденного контекста, значит проект работает корректно и достигает поставленной цели.
